# <img src="config/fish.png" alt="Logo" width="50"> YunDang-Chain 移动端全节点公有链 - 打造之旅


## <img src="config/fish.png" alt="Logo" width="50"> 特性
- 使用 NodeJS 构建
- 从第一行代码开始开发
- 全程在微信群、github上公布进展
- 所有人均有机会深度参与开发过程
- 每天核心攻关一个难点/开发任务
- 共识、存储、网络等所有环节全部实现在移动端
- 适配移动端展现 (Mobile First)
